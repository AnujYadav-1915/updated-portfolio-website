# Personal Portfolio

This is the repository for my personal developer portfolio, built with React.

It showcases my latest projects, technical skills, and background in software engineering, specifically focusing on backend systems, real-time architectures, and AI-backed applications.

## Tech Stack

- **Frontend:** React (Create React App)
- **Styling:** CSS
- **Deployment:** GitHub Pages (`gh-pages`)

## Local Setup

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/AnujYadav-1915/updated-portfolio-website.git
   cd updated-portfolio-website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

## Deployment

This portfolio is automatically deployed to GitHub Pages. To deploy a new version:

```bash
npm run deploy
```
